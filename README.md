Description:
    -The function's prototype is: char *get_next_line(int fd).
    -fd is the file descriptor of the file you want to read.
    -get_next_line will return a line with a '\n' except if the end of the file was reached and does
    not end with a '\n'.

Instructions:
    You should include all 3 files in your folder and include get_next_line.h in your file.
