This activity has been created as part of the 42 curriculum by ylau-sim.

Description:
    This project includes 43 functions, a makefile and a libft.h file. The makefile should compile all
the functions into libft.a

Instructions:
    Use the command make to compile all files into libft.a. You should then include libft.h into your c file and put libft.a and libft.h into the folder. You can use the commands clean and fclean to 
remove .o and .a files.

Resources:
    I used libfttester by tripouille to check the validity of the functions.
